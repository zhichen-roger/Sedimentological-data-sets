1数据集介绍：比赛提供根据在抽取代码文件夹下生成的命名体识别数据集，本数据集包含，时间，地点和物质三类标签分别用<B-tim,I-tim>,<B-nat,I-nat>,<B-geo,I-geo>表示，其中物质信息包含但不仅限于：群组、生物、地层等（例如群组单词Formation，还需识别出前面A或B关键词，形成有研究价值关键词，A Formation或 A B Formation）。时间信息包含但不仅限于：时期（例如白垩纪时期，还需识别出前面修饰词形成有研究价值关键词，早白垩纪时期）、MA（百万年）等。地点信息包含但不仅限于：山脉等。数据集共计274291个标签。可自行分成训练集和测试集。

2.数据集生成说明：可自行设计生成数据集方法，或改良本赛题所提供的数据集生成方法。

3.输入输出说明：输入为文本数据可自行下载补充，但文本要为互联网公开语料，格式为PDF。输出为提取的关键信息，其中至少分成物质信息、时间信息、地点信息三种情况。

4.实验效果说明：本赛道已用BiLSTM-CRF命名体识别模型，最终各标签准确率均要大于基准模型准确率。
